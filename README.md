# Mila Henning CV Site

Static CV website for Mila Henning.

## Run

Open `Index.html` in a browser.

## Contact

The contact section is a static form. When a visitor clicks the send button, their default mail application opens with a prefilled email to Mila.

## Files

- `Index.html` - page structure
- `style.css` - styling
- `script.js` - page interactions and mailto contact behavior
- `Screenshot 2026-04-21 162547.png` - profile image
